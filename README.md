## 🤖 Introduction

This is a sushi restaurant website showcasing popular sushi, drinks, and an “About Us” section.  
It is fully responsive and includes smooth CSS animations for an engaging user experience.  

---

## ⚙️ Tech Stack

- HTML 5
- CSS 3
- Vite (for fast development)
- AOS (Animate On Scroll library)

---

## 🔋 Features

- **Responsive Design:** Works on all screen sizes using media queries.  
- **CSS Animations:** Smooth scroll and element animations.  
- **Modular CSS:** Section-based CSS for easier maintenance.  
- **Reusable Components:** Navigation, Hero, About, Menu, Trending, Newsletter, Footer.  

Sections included:  
- Navigation Bar  
- Hero Section  
- About Us  
- Popular Food & Trending Sushi  
- Trending Drinks  
- Newsletter Subscription  
- Footer
